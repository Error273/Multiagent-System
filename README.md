# Multiagent-System

Выполнено в рамках интенсива «Agents Week», ШАД

Система для помощи выбора товара в интернет магазине техники на
основе ReAct с использованием инструментов и памяти.


Реализованы агенты с использованием tools и памяти, а так же центральный оркестратор, координирующий работу четырёх специализированных
агентов (Retriever, Pros, Cons, Ranker) при решении составных задач.
